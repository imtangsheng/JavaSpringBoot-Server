
## gradle
- 在解压Gradle压缩包后,不要直接将bin文件夹添加到PATH环境变量中。

- 修改或创建一个GRADLE_HOME环境变量,值为Gradle解压后的安装目录路径。

- 将安装目录下的bin文件夹路径追加到PATH变量value值中。%GRADLE_HOME%\bin
